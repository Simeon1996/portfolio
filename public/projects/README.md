## Tips
- Prefer static screenshots over GIF/video for faster loading.
- Supported formats: `.png`, `.jpg`, `.webp`, `.svg`.
- Recommended dimensions: 1280x720 or 1600x900.
- Name files by feature or section (for example `citation-panel.svg`).
